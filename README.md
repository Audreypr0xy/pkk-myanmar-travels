# PKK Myanmar Travels & Tours 🌏

> **"Your Satisfaction is Our Victory"**

A modern, responsive travel agency website built for **PKK Myanmar Travels & Tours** — offering tour packages across Asia, Europe, pilgrimage destinations, cruise trips, and more.

## ✨ Features

- **Home Page** — Hero banner, services, tour packages, destinations, testimonials & contact
- **About Us Page** — Company story, values, team, and milestone timeline
- **Asia Tour Packages** — Japan, South Korea, China, Vietnam, Cambodia, Philippines & more
- **Europe Tour Packages** — Austria, Germany, Netherlands & more
- **Signature Tours** — Kashmir, cruise trips & special packages
- **Multi-page routing** — React Router with smooth navigation
- **Fully responsive** — Mobile & desktop friendly
- **Animated UI** — Framer Motion transitions throughout

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Audreypr0xy/pkk-myanmar-travels.git

# Navigate to project folder
cd pkk-myanmar-travels

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
